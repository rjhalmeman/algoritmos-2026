Exercícios para aulas

1) Faça um programa, usando HTML e JS que lê um nome completo e imprime esse nome em formato inglês. Por exemplo, Ayrton Senna da Silva torna-se: Silva, Ayrton Senna da. Deve-se criar uma função chamada nomeEmIngles.
2) Faça um programa que leia uma palavra e mostre-a invertida utilizando um laço while. Deve-se criar um função chamada inverter para realizar o processo.
3) Faça um programa que leia uma palavra e mostre-a invertida utilizando um laço for.
4) Faça um programa que verifique se uma palavra é um palíndromo.
5) Faça um programa que leia uma frase e substitua todas as letras “a” por “@”;
6) Faça um programa que leia uma frase e monte outra frase contendo apenas as vogais. (usar laço while). Por exemplo, a frase: a casa caiu => aaaaiu
7) Faça um programa que leia uma frase e monte outra contendo apenas as consoantes (usar laço for). Por exemplo, a frase: a casa caiu => csc
8) Faça um programa que leia uma palavra e gere uma nova palavra com todas as letras que são vogais em maiúsculo. Por exemplo, a frase: a casa caiu => A cAsA cAIU.
9) Faça um programa que leia uma frase e crie uma nova string removendo todos os espaços.
10) Faça um programa que leia uma frase e crie uma nova string removendo todos os espaços que estiverem "sobrando".
